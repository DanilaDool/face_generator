# face_generator
Программа генерирует случайную рожицу из символов, и помещяет это в отдельный файл. Если программа генерирует больше одной рожицы в день, она помещяет это в один файл.
Пример:  / --- \
        | O   o |
        \   v   /
         \  O  /
------------------------------------
         / \_/ \
        | *   * |
        \   v   /
         \  O  /
------------------------------------
